# SPEC 19 – Native und Special Provider Adapter

## Ziel
ForgeFrame bekommt echte Spezialadapter für nicht generisch abbildbare Provider.

## Klassen
`anthropic_messages`, `gemini_generate_content`, `bedrock_converse`, `azure_openai_v1`, `vertex_openai`, `perplexity_sonar`, `minimax_anthropic`, `microsoft_agent_endpoint`, `local_openai_extended`.

## Anforderungen
OpenAI Request Mapping, Native Request Mapping, Native Error Mapping, Streaming Mapping, Tool Mapping, Capability-/Execution-/Policy-Profil, Evidence je Pfad.
