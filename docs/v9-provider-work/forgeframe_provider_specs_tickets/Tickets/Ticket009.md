# Ticket009 – OpenAI Compatibility Golden Corpus anlegen

**Priorität:** P0

Golden-Corpus für Chat, Responses, Streaming, Tools, Errors, Unsupported und Signoff erzeugen.
## Definition of Done
- Backend-Code oder Registry umgesetzt.
- Admin-/UI-Sicht ergänzt, falls operatorisch relevant.
- Migrationen ergänzt, falls Persistenz geändert wird.
- Unit-/Contract-Tests ergänzt.
- Kein Status wird grüner dargestellt als Evidence erlaubt.
- Keine Live-Fertigstellung ohne Credentials behaupten.
