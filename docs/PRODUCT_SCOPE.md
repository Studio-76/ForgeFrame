# PRODUCT_SCOPE

## Was ForgeGate ist
ForgeGate ist ein Smart AI Gateway, das Runtime-Ausführung, Provider-Anbindung und administrative Steuerung in einer konsistenten Plattform zusammenführt.

## Was ForgeGate in Phase 1 bewusst noch nicht ist
- Kein produktiv lauffähiges Gateway mit voller Logik.
- Kein fertiges OAuth-System.
- Kein fertiges Multi-Provider-Orchestrierungssystem.

## Produktachsen

### 1) AI Gateway
- Einheitlicher Entry Point für AI-Workloads.
- Routing- und Fallback-Strategien (später).

### 2) Provider Hub
- Adapter pro Provider.
- Normierte Provider-Fähigkeiten und Fehlerbilder.

### 3) Account Management
- Verwaltung von Konten und Berechtigungsbezügen.
- Trennung von Account-, Key- und Usage-Domänen.

### 4) Admin Platform
- UI und Admin-API für Betrieb, Konfiguration und Einsicht.

### 5) Runtime + Admin API
- Runtime API für Laufzeitaufrufe.
- Admin API für Betriebs- und Managementfunktionen.

## Feature-Phasen

### MVP
- Basale Runtime-Verarbeitung.
- Basis-Provider-Dispatch.
- Administrativer Kern für Provider/Konten/Keys.

### Phase 2
- Streaming- und Fallback-Härtung.
- Tool-Calling-Kompatibilität in Breite.
- Ausgebautes Monitoring und Usage-Transparenz.

### Später
- Erweiterte Policies.
- Erweiterte Routing-Intelligenz.
- Betriebsreife Multi-Provider-Strategien.
