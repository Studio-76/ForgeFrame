# Reference Material

Der Ordner `reference/` ist **ausschließlich Referenzmaterial**.

## Regeln
- Referenzdateien sind nicht produktiv.
- Es gibt keine produktiven Imports aus `reference/`.
- ForgeGate-Core wird neu in `backend/` und `frontend/` aufgebaut.

## Zweck
- Semantikvergleich
- Randfallanalyse
- Härtungswissen für spätere Neuimplementierung

## Hinweis
Der Referenzordner kann im Verlauf des Projekts reduziert, archiviert oder entfernt werden, sobald sein Nutzen erfüllt ist.
